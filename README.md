## 주차관리 프로그램 프로젝트

- 주차 차량 관리와 요금 정산을 위한 프로그램으로 일반 사용자는 입차와 출차, 정기권 차량 등록을 할 수 있다.
- 관리자는 주차 가능 구역 개수 지정 및 정기권 차량 관리를 할 수 있다.
- 기본적으로 주차 타워의 주차 시스템을 활용하며 관리자는 정기권 우선 주차대수를 확보할 수 있다.
